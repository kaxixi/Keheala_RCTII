# Intervention Materials

This folder contains a sample of the materials used in the digital adherence interventions evaluated in the Keheala RCT II trial (ClinicalTrials.gov `NCT04119375`). It is released alongside the trial's analysis code and de-identified data to help readers understand what the interventions looked like and to support replication research.

## Scope and licensing

The complete message library, TB-information content, and operational protocols — as well as the underlying Keheala platform software — are proprietary to **Keheala, Inc.** The contents of this folder are a curated sample, released under the [Creative Commons Attribution–NonCommercial 4.0 International license (CC-BY-NC-4.0)](LICENSE).

Complete materials are available from Keheala, Inc. on reasonable request for research purposes.

## File conventions

Every file in this folder opens with a short header indicating:

- **Scope** — either "Complete" (the entire content used during the trial) or "Sample of N of approximately M" (a representative selection).
- **Source** — where the content was extracted from (e.g., Keheala production platform, study team training manual, manuscript excerpt).
- **Date extracted** — when the content was pulled from its source.
